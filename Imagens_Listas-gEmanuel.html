Imagem

  • Imagem é a representação visual de alguma coisa
  • Imagem digital é a representação de uma imagem em duas dimensão (bidimensional) usando números binários codificados.
  • Essas imagens, através de processamento eletrônico, podem ser armazenadas, transferidas, impressas ou reproduzidas
  • Formatos: .bmp, .tiff, .gif, .svg, .ico, .jpeg, .png, dentre outros.

Redimencionando a imagem

  1. Utilizando-se de porcentagem

    2. Código: <img src = "img/imagem_arco_iris.jpg" width = "25%" height = "30%" alt = "Paisagem com arco-íris">



  2. Utilizando-se de px (pixels)

    3. Código: <img src = "img/imagem_arco_iris.jpg" width = "700px" height = "400px" alt = "Paisagem com arco-íris">

    4. Código: <img src = "img/imagem_arco_iris.jpg" width = "450px" height = "450px" alt = "Paisagem com arco-íris">

Listas

As listas podem ser de três tipos:

  • Não-ordenadas
  • Ordenadas
  • Detalhada

Listas Não Ordenadas

Utiliza-se da tag <ul> para a marcação da lista e da tag <li> para a marcação dos itens

Sintaxe:

<ul>
<li>item 1</li>
<li>item 2</li>
<li>item 3</li>
<li>item n</li>
</ul>

Exemplo: alguns livros da Colleen Hoover
  • É assim que acaba
  • Novembro, 9
  • Todas as suas (im)perfeições
  • Confesse
  • Talvez um dia

Código:

<ul>
<li>É assim que acaba</li>
<li>Novembro, 9</li>
<li>Todas as suas (im)perfeições</li>
<li>Confesse</li>
<li>Talvez um dia</li>
</ul>


Listas Ordenadas

Utiliza-se da tag <ol> para a marcação da lista, da tag <li> para a marcação dos itens e do atributo type, que define em qual elemento e qual o tipo de caractere para os marcadores

Sintaxe:

<ol>
<li>item 1</li>
<li>item 2</li>
<li>item 3</li>
<li>item n</li>
</ol>

Exemplo: sumário de um livro
  1. Introdução
  2. Conceitos Iniciais
  3. Títulos e Parágrafos
  4. Imagens e Listas

Código:

<ol>
<li>Introdução</li>
<li>Conceitos Iniciais</li>
<li>Títulos e Parágrafos</li>
<li>Imagens e Listas</li>
</ol>

Atributo type

É um atributo que é responsável por controlar os marcadores.

Alguns exemplos de valores e de sua respectiva utilidade:

  • type = "1" --> define que os marcadores são numéricos e que eles começam no número 1

    1. 1° Item
    2. 2° Item

    <ol type = "1">
    <li>1° Item</li>
    <li>2° Item</li>
    </ol>

  • type = "a" --> define que os marcadores são letras e que eles começam na letra a minúscula

    1. 1° Item
    2. 2° Item

    <ol type = "a">
    <li>1° Item</li>
    <li>2° Item</li>
    </ol>

  • type = "A" --> define que os marcadores são letras e que eles começam na letra a maiscula

    1. 1° Item
    2. 2° Item

    <ol type = "A">
    <li>1° Item</li>
    <li>2° Item</li>
    </ol>



Listas Detalhadas

Utiliza-se da tag <dl> para a marcação da lista, da tag <dt> para a marcação dos itens e da tag <dl> que define o detalhamento dos itens

Sintaxe:

<dl>
<dt>item 1</dt>
<dd>detalhamento do item 1</dd>
<dt>item 2</dt>
<dd>detalhamento do item 2</dd>
<dt>item 3</dt>
<dd>detalhamento do item 3</dd>
</dl>

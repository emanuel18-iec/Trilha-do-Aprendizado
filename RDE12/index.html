Gabriel Emanuel


O modelo das caixas em CSS, são responsáveis por descrever os boxes gerados pelos elementos HTML, detalhando as opções de ajuste de margens, bordas, padding e conteúdo para cada elemento.

Entender o box model, cujo é fundamental para construção de layouts em sites, pode ser compreendido através do modelo genérico representado a seguir:

Neste caso, o Content representa um elemento HTML qualquer, seja ele um paragráfo, um título, uma imagem ou qualquer outra coisa. Para o HTML, todos os elementos são vistos como uma caixa, por mais que isso não seja definido, ela existe e envolve o conteúdo.

Após o Content, o padding seria uma espécie de preenchimento, sendo definida a distância entre o conteúdo e uma borda, a qual veremos a seguir.

Se divide em:

  • padding-top: Preenchimento superior

  • padding-right: Preenchimento direito

  • padding-bottom: Preenchimento inferior

  • padding-left: Preenchimento esquerdo


Na sequência, define-se o border, inclusive se este aparece ou não. Da mesma forma, tem-se as quatro subdivisões/medidas:

  • border-top: Borda superior

  • border-right: Borda direito

  • border-bottom: Borda inferior

  • border-left: Borda esquerdo


Por fim, define-se a margem, que também se subdivide em

  • margin-top: Margem superior

  • margin-right: Margem direito

  • margin-bottom: Margem inferior

  • margin-left: Margem esquerdo

Para se definir todas essas propriedades, lista-se os seus valores em HORÁRIA, iniciando-se pelo "top" (superior) indo até o "left" (esquerdo).

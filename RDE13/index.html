Inicialmente, vamos começar alterando algumas propriedades no body, para que estas sejam determinadas de forma padrão

Vamos começar definindo um font-family. Escolha uma ao seu gosto. Usarei como exemplificação, a 'Gill Sans', 'Gill Sans MT', Calibri, 'Trebuchet MS', sans-serif

Além disso, vamos aplicar uma cor de fundo degradê para o nosso site. Para isso, usaremos a propriedade background-image, seguido do valor "linear-gradient". Ou seja, será aplicado um gradiente linear ao nosso conteúdo.

O primeiro parâmetro que podemos definir, é a direção do gradiente, utilizando-se do prefixo "to" seguido da direção. Por exemplo, para indicar que o gradiente é para baixo, usamos o comando "to bottom"

Em seguida, vamos definir os parâmetros referente as cores que serão aplicadas nesse gradiente.

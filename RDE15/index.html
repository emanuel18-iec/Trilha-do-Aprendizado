No entanto, essas informações não estão sendo enviadas em si, mesmo com o botão sendo para isso. Para isso, usaremos, na tag <form> do próprio HTML, o atributo  action  , que vai receber as intruções do que faz quando o botão de envio for acionado. Geralmente, utiliza-se uma referencia para um outro arquivo com extensão php, como por exemplo:  action = "cadastro.php"  .

Além disso, determina-se um  name  , que é uma espécie de identificação muito importante para a linguagem PHP.


Outro atributo importante, é o  method  , que será a forma com o qual esses dados serão repassados para o arquivo definido no  action  . Podem ser do tipo  method = "post"  e do tipo  method = "get"  . Quando o valor é  "get"  , quer dizer que os dados serão visíveis pela URL do site. Ao contrário do método  "post"  , que não exibe.
